## 1. 产品概述

| 项目       | 内容                                                                       |
| ---------- | -------------------------------------------------------------------------- |
| 产品名称   | 问玄东方·法师工作台                                                         |
| 产品定位   | 面向已入驻平台寺院法师/道士/活佛的移动端工作管理工具                         |
| 核心价值   | 帮助法师高效管理预约、日程、收入与用户沟通，提升接单效率与服务质量            |
| 使用对象   | 已通过平台资质审核并入驻的寺院法师、道士、活佛等宗教从业者                   |
| 端类型     | iOS App（优先）/ 微信小程序（二期）                                         |
| 登录方式   | 手机号 + 短信验证码（需先通过平台资质审核后方可登录）                         |

### 产品愿景

为寺院法师提供专业、便捷的移动工作管理平台，实现接单、日程、沟通、收入的一站式管理，让法师专注于弘法利生，平台负责效率赋能。

### 与C端App的关系

法师工作台与C端用户App共享同一套设计语言（红金火土色彩体系），是C端App中「预约」「咨询」等功能的服务端对应。C端用户发起的预约请求、咨询消息，均会在法师工作台对应的模块中呈现。

| 属性       | 描述                                                   |

## 4. 功能模块与需求

### 4.1 登录页

**需求编号**: FS-001  
**页面层级**: 入口页（无底部Tab栏）  
**页面路径**: `/login`

#### 功能描述

法师通过手机号+验证码登录工作台。只有已通过平台资质审核的法师方可登录，未审核的账号会提示联系寺院管理员。

#### 页面布局

| 区域           | 内容                                                         |
| -------------- | ------------------------------------------------------------ |
| 顶部Logo区     | 问玄东方Logo + "法师工作台" 品牌标题                          |
| 手机号输入     | 手机号输入框（+86前缀），placeholder："请输入手机号"            |
| 验证码区       | 验证码输入框（6位数字） + "获取验证码" 按钮（60秒倒计时）       |
| 登录按钮       | 朱砂红主按钮 "登录"                                           |
| 协议区         | 勾选框 + "我已阅读并同意《法师入驻协议》和《隐私政策》"        |
| 底部提示       | "如未入驻平台，请联系寺院管理员" 灰色辅助文字                    |

#### 交互流程

```
用户输入手机号
 └-> 点击"获取验证码"
     └-> 按钮变为60秒倒计时
 └-> 输入6位验证码
 └-> 点击"登录"
     ├-> [成功] 首次登录 -> 进入工作台首页
     ├-> [成功] 非首次登录 -> 进入工作台首页
     └-> [失败] 提示："该手机号未通过资质审核，请联系寺院管理员"
```

#### 数据字段

| 字段           | 类型     | 校验规则                   |
| -------------- | -------- | -------------------------- |
| 手机号         | string   | 11位手机号，以1开头        |
| 验证码         | string   | 6位数字                    |
| 协议勾选       | boolean  | 必须勾选才能登录           |

#### 业务规则

| 规则             | 描述                                                   |
| ---------------- | ------------------------------------------------------ |
| 资质审核         | 只有审核通过状态的法师账号才允许登录                    |
| 验证码有效期     | 验证码5分钟内有效                                       |
| 发送频率限制     | 同一手机号60秒内不可重复发送                             |
| 自动登录         | 登录成功后Token有效期30天，期间免登录                    |

**页面层级**: 一级（Tab页，含底部Tab栏）

| 区域             | 内容描述                                                     |
| ---------------- | ------------------------------------------------------------ |

| 序号 | 入口名称   | 图标描述      | 跳转目标                    |

样式：
- 图标: 24px，`#C8A96E`
- 文字: 11px，`#C5B097`
- 间距: 等分4列，图标在上、文字在下
- 点击态: 图标变为 `#C45A3C`

**今日预约列表**

每张预约卡片：

| 字段       | 类型     | 示例值             | 样式说明                     |
| ---------- | -------- | ------------------ | ---------------------------- |
| 时间       | 文本     | 14:00-15:00        | 14px白色粗体                 |
| 服务类型   | 文本     | 祈福消灾           | 13px `#C5B097`              |
| 用户昵称   | 文本     | 信众·张先生         | 13px `#C5B097`              |

- 左侧：时间区间（垂直居中，白色粗体）

- 休息模式开关: Toggle组件，开启后停止接收新预约

#### 交互行为

| 触发元素       | 交互行为                                         |
| -------------- | ------------------------------------------------ |

## 6. 用户核心流程

### 6.1 法师确认预约流程

```
工作台首页
 └-> 查看今日预约卡片
     └-> 点击进入预约详情
         ├-> [确认] 内联确认 -> 状态: 已确认 -> 通知用户
         └-> [拒绝] 选择原因 -> 确认拒绝 -> 状态: 已拒绝 -> 通知用户
```

### 6.2 法师服务履约流程

```
预约列表 (Tab 2)
 └-> 已确认Tab
     └-> 点击预约卡片 -> 预约详情
         └-> 点击"开始服务" -> 状态: 进行中
             └-> 服务完成 -> 点击"完成服务" -> 状态: 已完成
                 └-> 提示用户评价
                     └-> 收到评价通知 (Tab 3 消息中心)
```

### 6.3 日程管理流程

```
工作台首页
 └-> 快捷入口"设置日程"
     └-> 日历管理页
         ├-> 点击某天 -> 查看当天时段 -> 编辑时段
         ├-> "批量设置可约时段" -> 选择周几/时间/月份 -> 确认
         └-> "设置休息日" -> 选择日期/原因 -> 确认
```

### 6.4 法师回复用户咨询流程

```
消息中心 (Tab 3)
 └-> 咨询Tab
     └-> 对话列表
         └-> 点击某对话 -> 对话详情页
             └-> 输入消息 -> 发送
             └-> [可选] 发送图片
```

### 6.5 收入提现流程

```
个人中心 (Tab 4)
 └-> 收入管理 -> 收入管理页
     ├-> 查看本月收入/累计收入/可提现金额
     ├-> 浏览收入明细列表
     └-> 点击"提现" -> 提现弹窗
         └-> 输入金额 -> 选择提现方式 -> 确认
             └-> 等待审核/到账
```

## 8. 权限与安全

### 8.1 角色权限矩阵

| ---------------- | -------- | --------- | --------- |

| 确认/拒绝预约    | 仅自己    | 仅自己    | 全寺院    |
| 管理个人日程     | 仅自己    | 仅自己    | 仅自己    |
| 修改个人资料     | 仅自己    | 仅自己    | 仅自己    |
| 设置服务定价     | 仅自己    | 仅自己    | 仅自己    |
| 查看收入         | 仅自己    | 仅自己    | 可选      |
| 提现             | 仅自己    | 仅自己    | 仅自己    |
| 管理寺院法师     | 否       | 否       | 是        |
| 设置法会日程     | 否       | 否       | 是        |
| 查看全院数据     | 否       | 否       | 是        |

### 8.2 安全措施

| 措施             | 描述                                                     |
| ---------------- | -------------------------------------------------------- |
| 登录验证         | 手机号 + 短信验证码，Token有效期30天                     |
| 自动锁屏         | App进入后台超过5分钟需重新验证（指纹/面容/验证码）         |
| 数据加密         | 所有API通信使用HTTPS + TLS 1.2+                           |
| 手机号脱敏       | 用户手机号在法师端展示为脱敏格式（138****5678）            |
| 操作日志         | 关键操作（确认/拒绝预约、提现、修改资料）记录操作日志      |
| 防刷机制         | 验证码发送频率限制、登录失败次数限制                      |

## 10. 与C端App的数据联动

### 10.1 数据流向

| C端用户操作          | 法师工作台响应                              |
| ------------------- | ------------------------------------------- |
| 用户提交预约         | 工作台首页更新"今日预约数" + 新预约通知推送     |
| 用户取消预约         | 预约列表中状态变更 + 取消通知推送              |
| 用户发送咨询消息      | 消息中心咨询Tab新增未读消息 + 推送提醒          |
| 用户完成评价         | 评价管理新增评价 + 新评价通知推送               |
| 用户支付功德金       | 收入管理新增记录 + 收入到账通知推送             |

| 法师工作台操作       | C端用户响应                                  |
| ------------------- | ------------------------------------------- |
| 法师确认预约         | 用户收到"预约已确认"通知                        |
| 法师拒绝预约         | 用户收到"预约已拒绝"通知 + 退还功德金           |
| 法师开始服务         | 用户收到"服务已开始"通知                        |
| 法师完成服务         | 用户收到"服务已完成，请评价"通知                 |
| 法师设置休息         | 用户端法师主页显示"休息中"状态                  |
| 法师修改资料         | 资料审核通过后用户端同步更新                    |

### 10.2 共享数据字段

| 数据字段       | C端用途                     | 法师端用途                  |
| -------------- | --------------------------- | --------------------------- |
| 法师头像       | 师傅列表/主页展示           | 工作台/个人中心展示          |
| 法号           | 师傅列表/主页展示           | 工作台/个人中心展示          |
| 寺院信息       | 师傅主页展示               | 个人资料/预约详情展示        |
| 服务类型与价格 | 预约下单页选择             | 服务定价管理/收入明细        |
| 预约状态       | 预约记录展示               | 预约列表/详情管理            |
| 评价内容       | 师傅主页评价Tab展示        | 评价管理展示                 |
| 专长标签       | 师傅主页专长展示           | 专长管理编辑                 |
