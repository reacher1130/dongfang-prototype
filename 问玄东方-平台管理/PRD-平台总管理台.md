## 1. 产品概述

| 项目     | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| 产品名称 | 问玄东方·平台管理台（平台总管理台）                           |
| 使用对象 | 平台运营人员、客服、财务、技术管理员                         |
| 端类型   | Web端（PC浏览器，分辨率 ≥ 1280×720）                         |
| 登录方式 | 账号密码 + 验证码 + 二次验证（TOTP / 短信验证码，可按角色配置）|

### 1.2 核心目标

1. 为平台运营团队提供高效、统一的后台管理工具
2. 覆盖寺院/法师全生命周期管理
3. 实现内容审核、财务结算、营销活动的闭环运营
4. 通过角色权限体系保障数据安全与操作合规

### 2.3 权限说明

- **○（仅查看）**：可查看列表和详情，但不可进行编辑、审核、删除等写操作
- **✓（完整权限）**：包含查看、新增、编辑、审核、删除等全部操作
- **✗（无权限）**：不可访问该模块，侧边栏不显示对应菜单
- 超级管理员权限不可被降级，至少保留一个超级管理员账号

### 4.1 登录页

#### 4.1.1 页面说明

平台管理台入口页面，采用居中卡片式布局，背景使用侧边栏色 `#1C1210` 渐变，登录卡片白色背景。

#### 4.1.2 页面元素

| 元素         | 类型       | 说明                                          |
| ------------ | ---------- | --------------------------------------------- |
| 平台Logo     | 图片       | 问玄东方Logo，尺寸 60×60px，居中顶部           |
| 平台名称     | 文字       | "问玄东方·平台管理台"，字号20px，主文字色      |
| 账号输入框   | 输入框     | placeholder "请输入账号"，前缀图标：用户      |
| 密码输入框   | 寓言输入框 | placeholder "请输入密码"，前缀图标：锁，支持显示/隐藏切换 |
| 验证码输入框 | 输入框+图片 | 4位字母数字验证码，点击图片刷新，120×40px       |
| 二次验证     | 输入框     | 仅当账号开启二次验证时显示，TOTP码或短信验证码  |
| 登录按钮     | 主按钮     | 色值 `#C45A3C`，宽度100%，"登 录"              |
| 忘记密码链接 | 文字链接   | 色值 `#C8A96E`，字号13px                       |

#### 4.1.3 交互规则

1. 输入校验：
   - 账号：必填，长度4-32位
   - 密码：必填，长度6-32位
   - 验证码：必填，不区分大小写
   - 二次验证：条件必填
2. 错误提示：输入框下方红色文字，行内显示
3. 登录失败5次后锁定账号30分钟，显示剩余锁定时间
4. 登录成功后跳转至数据大盘页面
5. Session有效期8小时，超时自动跳转登录页

#### 4.1.4 安全要求

- 密码传输加密（RSA）
- 验证码有效期5分钟
- 登录日志记录IP、设备指纹、时间
- 支持单点登录限制（同一账号仅允许一个终端在线）

### 4.2 数据大盘

#### 4.2.1 页面说明

平台运营核心数据概览页，登录后的默认首页。帮助运营人员快速了解平台整体运营状态。

#### 4.2.2 核心指标卡片

4个大数字卡片横排排列，等宽分布，间距24px。

| 卡片       | 主指标          | 辅指标            | 趋势指示         |
| ---------- | --------------- | ----------------- | ---------------- |
| 用户       | 总用户数        | 今日新增          | 较昨日 ↑/↓ 百分比 |
| 订单       | 总订单数        | 今日订单数        | 较昨日 ↑/↓ 百分比 |
| 寺院       | 活跃寺院数      | 待审核申请数      | 较昨日 ↑/↓ 百分比 |
| 收入       | 今日收入（元）  | 较昨日增长        | ↑ 绿色 / ↓ 红色   |

**卡片规格**：
- 背景：白色，圆角8px，box-shadow
- 主指标：字号28px，字重700，主文字色 `#333`
- 辅指标：字号14px，辅助文字色 `#666`
- 趋势：字号13px，增长绿色 `#4CAF50` / 下降红色 `#E53935`
- 右上角：小图标 + 背景色块

#### 4.2.3 用户增长趋势（折线图）

- 标题："用户增长趋势（近30天）"
- X轴：日期（近30天），格式 MM-DD
- Y轴：新增用户数
- 两条折线：
  - 实线 `#C45A3C`：注册用户数
  - 虚线 `#C8A96E`：活跃用户数
- 鼠标悬浮显示具体数值
- 支持切换时间范围：近7天 / 近30天 / 近90天

#### 4.2.4 订单量趋势（折线图）

- 标题："订单量趋势（近30天）"
- X轴：日期
- Y轴：订单数
- 三条折线：
  - 实线 `#C45A3C`：总订单
  - 虚线 `#C8A96E`：已完成订单
  - 点线 `#999999`：退款订单
- 鼠标悬浮显示具体数值

#### 4.2.5 服务类型分布（饼图）

- 标题："服务类型分布"
- 数据项及对应颜色：

| 服务类型   | 颜色     |
| ---------- | -------- |
| 祈福       | `#C45A3C`|
| 供灯       | `#C8A96E`|
| DIY手串    | `#8B6F4E`|
| 开光       | `#A0522D`|
| 法事       | `#6B8E6B`|
| 其他       | `#CCCCCC`|

- 鼠标悬浮显示：服务类型名称 + 数量 + 占比百分比
- 右侧显示图例

#### 4.2.6 实时动态

- 标题："实时动态"
- 列表展示最近10条动态，每条一行
- 字段格式：`[时间] [类型图标] [内容描述]`

| 动态类型   | 图标 | 示例内容                     |
| ---------- | ---- | ---------------------------- |

- 自动刷新：每30秒轮询更新
- 超过10条后滚动，最新在顶部

#### 4.2.7 地域分布（柱状图）

- 标题："用户地域分布 TOP10"

## 9. 附录

### 9.1 状态流转汇总

**寺院状态：**
```
待完善 → 待审核 → 正常 → 已封禁
                ↓         ↑
              驳回       解封
```

**法师认证状态：**
```
未认证 → 待审核 → 已认证 → 已封禁
            ↓         ↑
          驳回       解封
```

**结算状态：**
```
待结算 → 待审核 → 待打款 → 已完成
            ↓
          驳回 → 待结算（修正后重新发起）
```

**举报状态：**
```
待处理 → 已处理 / 已驳回
```
