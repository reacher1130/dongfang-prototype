## 1. 产品概述

| 项目     | 内容                                                           |
| -------- | -------------------------------------------------------------- |
| 产品名称 | 东方玄学 · 商城管理台                                           |
| 产品定位 | 面向商城运营人员的Web端后台管理系统                               |
| 核心价值 | 高效管理商品上架与库存、处理DIY手串定制订单、追踪物流与售后，实现商城业务闭环运营 |
| 使用对象 | 商城运营人员、仓储物流人员、客服人员                               |
| 端类型   | Web端（PC浏览器）                                               |

### 1.1 产品愿景

打造一个界面清晰、操作高效的商城管理后台，让运营人员能够快速处理商品管理、DIY手串定制订单、材料库存和物流发货等日常电商运营工作，支撑"东方玄学"平台商城业务的规模化运转。

### 1.2 目标用户画像

| 用户角色   | 职责范围                                                     | 核心诉求                                               |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------------ |
| 运营专员   | 商品上下架、分类管理、数据报表查看                              | 快速上架商品、灵活管理分类、实时掌握销售数据              |
| 仓储人员   | 库存管理、材料出入库、打包发货                                  | 准确管理库存数量、快速处理发货、及时预警低库存              |
| 客服人员   | 订单查询、退换货处理、物流跟踪                                  | 快速定位订单、高效处理售后问题                             |
| DIY工坊人员 | DIY订单制作、状态更新、材料消耗记录                              | 清晰了解订单需求、追踪制作进度、记录材料消耗               |

## 3. 页面详细规格

### 3.1 登录页

**需求编号**: SC-001
**页面路径**: `/login`
**功能描述**: 管理台统一登录入口，支持账号密码登录。

#### 页面布局

| 区域         | 内容描述                                                       |
| ------------ | -------------------------------------------------------------- |
| 左侧品牌区   | 平台品牌展示区：东方玄学Logo + 品牌Slogan + 禅意背景装饰图       |
| 右侧登录区   | 登录表单卡片                                                   |

#### 登录表单字段

| 字段       | 类型   | 校验规则                                        | 占位文本       |
| ---------- | ------ | ----------------------------------------------- | -------------- |
| 账号       | 输入框 | 必填，4-20位字母/数字/手机号                    | 请输入账号      |
| 密码       | 输入框 | 必填，6-20位，支持显示/隐藏切换                  | 请输入密码      |
| 验证码     | 输入框 | 必填，4位图形验证码，点击图片刷新                | 请输入验证码    |
| 记住我     | 复选框 | 默认不选中，勾选后7天内免登录                    | -              |
| 登录按钮   | 按钮   | 主色 `#C45A3C`，圆角，加载状态显示spinner        | 登 录           |

#### 业务规则

| 规则             | 描述                                                       |
| ---------------- | ---------------------------------------------------------- |
| 登录失败         | 连续5次失败后锁定账号30分钟，弹出提示                       |
| 会话超时         | 无操作2小时后自动退出，跳转登录页                            |
| 密码加密         | 前端RSA加密传输，后端Bcrypt存储                              |
| 登录成功重定向   | 跳转至仪表盘首页                                            |

### 3.2 仪表盘

**需求编号**: SC-002
**页面路径**: `/dashboard`
**功能描述**: 运营数据总览，聚合今日核心指标、趋势图表和待办事项。

#### 今日数据卡片

4个数据卡片横向排列，每个卡片包含图标、标题、数值、同比变化标签。

| 卡片名称   | 数据字段    | 单位   | 同比标签说明       | 点击跳转              |
| ---------- | ----------- | ------ | ------------------ | --------------------- |
| 今日订单   | todayOrders | 单     | 较昨日 +/-XX%      | 商品订单列表          |
| 待发货     | pendingShip | 单     | 较昨日 +/-XX单     | 待发货订单筛选视图     |
| 今日收入   | todayRevenue| 元     | 较昨日 +/-XX%      | 数据报表              |
| 退货申请   | returnApply | 单     | 新增XX单            | 退换货管理列表         |

- 卡片样式：白色圆角卡片，左侧圆形图标区域使用主色/强调色浅色背景，右侧数值使用18px粗体。

#### 近7天订单趋势

| 配置项       | 说明                                          |
| ------------ | --------------------------------------------- |
| 图表类型     | 面积折线图（Area Line Chart）                  |
| X轴          | 最近7天日期（MM-DD格式）                       |
| Y轴          | 订单数量                                       |
| 数据系列     | 订单总量（主色填充）、DIY订单量（金色虚线）     |
| 交互         | 悬浮显示具体数值的Tooltip                      |
| 时间范围切换 | 支持"近7天"/"近30天"/"本月"Tab切换             |

#### 热销商品 TOP5

| 排行项   | 数据字段     | 说明                                  |
| -------- | ------------ | ------------------------------------- |
| 排名     | rank         | 1-5名，前三名用主色/金/铜色标识        |
| 商品名称 | productName  | 最多显示12个字符，超出省略号           |
| 商品图片 | productImage | 40x40px缩略图                         |
| 销量     | salesCount   | 累计销售数量                           |
| 销售额   | salesAmount  | 累计销售金额（元）                     |

- 列表样式：序号 + 图片 + 名称 + 销量 + 销售额，横向柱状图可视化销量占比。

#### DIY订单统计

4个状态统计卡片，与C端DIY模块对应。

| 状态     | 颜色标识 | 说明                                   | 点击跳转          |
| -------- | -------- | -------------------------------------- | ----------------- |
| 待审核   | `#E8A838` | 用户提交DIY订单等待运营审核              | DIY订单(待审核)   |
| 制作中   | `#C45A3C` | 审核通过进入工坊制作阶段                | DIY订单(制作中)   |
| 待发货   | `#C8A96E` | 制作完成等待物流发货                    | DIY订单(待发货)   |
| 已完成   | `#5B8C5A` | 已发货且用户确认收货                    | DIY订单(已完成)   |

- 每个卡片显示数量和占全部DIY订单的百分比。

#### 待办事项

| 待办类型   | 优先级 | 描述                                             | 点击操作       |
| ---------- | ------ | ------------------------------------------------ | -------------- |
| 库存预警   | 高     | X件商品库存低于预警值                             | 跳转商品列表   |
| 退货待处理  | 中     | X件退货申请超48小时未处理                         | 跳转退换货管理 |
| DIY待审核  | 中     | X个DIY订单等待审核                                | 跳转DIY订单列表|
| 物流异常   | 低     | X个包裹超过72小时无物流更新                        | 跳转物流管理   |

### 3.3 商品管理列表

**需求编号**: SC-003
**页面路径**: `/product/list`
**功能描述**: 商品列表展示与管理，支持多维筛选、搜索和批量操作。

#### 页面布局

| 区域         | 内容描述                                                     |
| ------------ | ------------------------------------------------------------ |
| 页面标题     | "商品管理" + 面包屑导航                                      |
| 操作工具栏   | 搜索框 + 分类筛选 + 状态筛选 + 添加商品按钮 + 批量操作下拉    |
| 数据表格     | 商品列表表格                                                 |
| 分页器       | 页码 + 每页条数选择 + 总数显示                                |

#### 筛选条件

| 筛选项   | 类型     | 选项                                                   |
| -------- | -------- | ------------------------------------------------------ |
| 关键词   | 输入框   | 支持按商品名称/编号搜索，回车或点击搜索按钮触发         |
| 商品分类 | 下拉选择 | 全部 / 法器 / 手串 / 项链 / 配饰 / 文创 / 其他         |
| 商品状态 | 下拉选择 | 全部 / 已上架 / 已下架 / 草稿                          |
| 价格区间 | 范围输入 | 最小价格 - 最大价格                                      |
| 库存区间 | 范围输入 | 最小库存 - 最大库存                                      |

#### 数据表格字段

| 列名     | 字段           | 宽度   | 说明                                                  |
| -------- | -------------- | ------ | ----------------------------------------------------- |
| 复选框   | checkbox       | 40px   | 全选/单选，用于批量操作                                |
| 商品图片 | productImage   | 70px   | 50x50px正方形缩略图，点击可预览大图                     |
| 商品编号 | productId      | 100px  | 系统自动生成的唯一编号（SP开头+8位数字）                |
| 商品名称 | productName    | 自适应 | 最多显示20字符，超出显示省略号，悬浮显示完整名称        |
| 商品分类 | category       | 80px   | 分类标签，颜色区分                                      |
| 价格     | price          | 100px  | "原价"删除线 + "售价"粗体，双行显示                     |
| 库存     | stock          | 80px   | 当前库存/预警值，低于预警值红色高亮                      |
| 销量     | salesCount     | 80px   | 累计销售数量                                           |
| 状态     | status         | 80px   | 已上架(绿色)/已下架(灰色)/草稿(黄色) 状态标签           |
| 排序     | sortOrder      | 70px   | 数字，支持拖拽排序                                     |
| 创建时间 | createdAt      | 120px  | YYYY-MM-DD HH:mm格式                                  |
| 操作     | actions        | 150px  | 编辑 / 上架或下架 / 删除 / 预览                          |

#### 操作按钮说明

| 操作       | 条件               | 确认提示                          |
| ---------- | ------------------ | --------------------------------- |
| 编辑       | 所有状态           | 无，跳转编辑页                    |
| 上架       | 已下架/草稿        | 弹窗确认"确定上架该商品?"         |
| 下架       | 已上架             | 弹窗确认"确定下架该商品?"         |
| 删除       | 草稿状态           | 二次确认弹窗"确定删除?删除后不可恢复" |
| 预览       | 所有状态           | 打开新窗口展示商品详情页(C端视角)   |

#### 批量操作

| 操作       | 描述                               |
| ---------- | ---------------------------------- |
| 批量上架   | 选中商品批量切换为上架状态          |
| 批量下架   | 选中商品批量切换为下架状态          |
| 批量修改分类 | 选中商品统一修改分类               |
| 批量导出   | 导出选中商品为Excel文件             |

#### 业务规则

| 规则             | 描述                                                         |
| ---------------- | ------------------------------------------------------------ |
| 默认排序         | 按创建时间倒序                                               |
| 每页条数         | 默认20条/页，可选10/20/50/100                                |
| 上架条件         | 商品必填信息完整（名称/分类/价格/库存/至少1张主图/详情描述） |
| 上架通知         | 商品上架后，已收藏该商品的用户收到推送通知                    |
| 库存预警阈值     | 默认预警值为10，可单品设置                                    |

### 3.4 商品编辑

**需求编号**: SC-004
**页面路径**: `/product/edit/:id`（编辑）/ `/product/add`（新增）
**功能描述**: 商品信息的添加与编辑，采用分步骤Tab布局。

#### 页面布局

| 区域         | 内容描述                                                     |
| ------------ | ------------------------------------------------------------ |
| 页面标题     | "添加商品"/"编辑商品" + 面包屑导航                           |
| Tab导航      | 基本信息 / 价格库存 / 商品详情 / 物流设置  4个Tab           |
| 内容区       | 当前Tab对应的表单内容                                        |
| 底部操作栏   | 保存草稿 / 提交上架（固定在底部）                             |

#### 3.4.1 基本信息 Tab

| 字段         | 类型       | 必填 | 校验规则                                          | 说明                    |
| ------------ | ---------- | ---- | ------------------------------------------------- | ----------------------- |
| 商品名称     | 输入框     | 是   | 2-50个字符                                       | 不可与已有商品名称重复    |
| 商品编号     | 输入框     | 否   | 字母+数字，8-20位                                | 为空则系统自动生成       |
| 商品分类     | 级联选择   | 是   | 一级分类 + 二级分类                               | 分类数据来自分类管理     |
| 品牌         | 输入框     | 否   | 2-20个字符                                       | 支持输入已有品牌或新增   |
| 材质         | 输入框     | 否   | 多选标签，支持自定义输入                           | 如：小叶紫檀、南红玛瑙    |
| 商品标签     | 标签选择   | 否   | 最多5个标签                                      | 从预设标签库选择          |
| 商品图片     | 图片上传   | 是   | 最多9张，单张不超过5MB，支持jpg/png/webp          | 第一张默认为主图，可拖拽排序 |
| 视频上传     | 视频上传   | 否   | 最多1个，不超过100MB，支持mp4格式                  | 商品展示视频              |

**图片上传规格**:

| 项目       | 要求                                    |
| ---------- | --------------------------------------- |
| 数量限制   | 最少1张，最多9张                         |
| 主图要求   | 建议尺寸800x800px，正方形                |
| 详情图    | 建议宽度750px，高度不限                  |
| 格式       | JPG / PNG / WebP                        |
| 大小限制   | 单张不超过5MB                             |
| 操作       | 支持拖拽排序、设置主图、删除             |

#### 3.4.2 价格库存 Tab

| 字段         | 类型     | 必填 | 校验规则                          | 说明                      |
| ------------ | -------- | ---- | --------------------------------- | ------------------------- |
| 原价         | 数字输入 | 是   | >=0.01，保留2位小数              | 划线价                    |
| 售价         | 数字输入 | 是   | >0，<=原价，保留2位小数           | 实际售价                  |
| 会员价       | 数字输入 | 否   | >0，<=售价，保留2位小数           | VIP会员专属价             |
| SKU编码      | 文本输入 | 是   | 唯一，格式：SP+分类码+4位流水号    | 库存管理单位              |
| 库存数量     | 数字输入 | 是   | >=0，整数                         | 当前可售库存              |
| 库存预警值   | 数字输入 | 否   | >=1，整数，默认10                 | 低于此值触发预警           |
| 计价单位     | 下拉选择 | 是   | 件 / 串 / 条 / 个 / 对           | 与分类关联的默认单位       |

**价格业务规则**:

| 规则             | 描述                                             |
| ---------------- | ------------------------------------------------ |
| 售价不得高于原价 | 系统校验，违反时提示"售价不能大于原价"              |
| 会员价不得高于售价 | 系统校验，违反时提示"会员价不能大于售价"           |
| 价格修改         | 修改已上架商品售价需审批（可选开关）               |

#### 3.4.3 商品详情 Tab

| 功能点     | 描述                                                   |
| ---------- | ------------------------------------------------------ |
| 富文本编辑器 | 支持文字排版、插入图片、表格、视频嵌入                    |
| 商品描述   | 详细介绍商品特点、功效、适用人群、使用说明等               |
| 注意事项   | 保养说明、禁忌事项等（可选）                             |

## 4. 业务流程

### 4.1 普通商品上架流程

```
运营专员登录
 └-> 商品管理 -> 添加商品
     └-> 填写基本信息（名称/分类/品牌/材质/图片）
         └-> 设置价格库存（原价/售价/会员价/SKU/库存）
             └-> 编辑商品详情（富文本/参数）
                 └-> 设置物流（重量/运费模板）
                     └-> 提交上架
                         └-> 校验通过 -> 商品上架
                         └-> 校验失败 -> 提示缺失信息
```

### 4.2 DIY手串订单处理流程

```
用户在C端提交DIY订单
 └-> [管理台] DIY订单列表 -> 待审核
     └-> 运营审核
         ├── 审核通过 -> 状态变为"制作中"
         │   └-> 工坊接收 -> 备料 -> 串珠 -> 质检
         │       └-> 完成制作 -> 状态变为"待发货"
         │           └-> 录入物流信息 -> 状态变为"已发货"
         │               └-> 用户确认收货 -> 状态变为"已完成"
         │                   └-> 自动扣减材料库存
         │
         └── 审核驳回 -> 填写原因 -> 通知用户修改
             └-> 用户重新提交 -> 再次审核
```

### 4.3 退换货处理流程

```
用户发起退货申请（C端）
 └-> [管理台] 退换货管理 -> 待处理
     └-> 客服审核凭证和原因
         ├── 同意退货
         │   └-> 等待用户寄回商品
         │       └-> 收到商品确认
         │           └-> 执行退款（原路退回）
         │               └-> 退款成功 -> 订单状态更新
         │
         └── 拒绝退货
             └-> 填写拒绝原因 -> 通知用户
```

### 4.4 库存管理流程

```
[管理台] 材料库管理
 └-> 查看库存列表
     ├── 库存充足 -> 正常
     ├── 库存预警 -> 仪表盘红色提醒
     │   └-> 运营联系供应商补货
     │       └-> 入库更新库存数量
     └── 缺货 -> 禁止DIY订单审核通过（材料不足提示）
```
