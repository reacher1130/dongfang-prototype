## 1. 产品概述

| 项目     | 内容                                                           |
| -------- | -------------------------------------------------------------- |
| 产品名称 | 问玄东方                                                       |
| 产品定位 | 一款融合佛教、道教传统文化与现代科技的移动互联网平台             |
| 核心价值 | 连接信众与寺院/法师，提供在线祈福、AI问事、DIY手串定制等一站式服务 |

### 产品愿景

将中国传统文化与移动互联网技术深度融合，打造一个可信赖的佛道教文化服务平台，让用户随时随地便捷地接触玄学文化、获取命理指导、体验传统仪式。

## 2. 目标用户画像

### 2.1 信众用户

| 属性     | 描述                                           |
| -------- | ---------------------------------------------- |
| 年龄区间 | 25-55岁                                        |
| 特征     | 对佛教/道教有信仰，经常去寺庙祈福、供养        |
| 痛点     | 寺院距离远、不知道哪个法师适合自己、预约不便    |
| 核心需求 | 在线查找寺院/法师、预约祈福法事、了解佛教知识  |
| 期望场景 | 不出门即可完成寺院预约、在线祈福供养            |

### 2.2 好奇用户

| 属性     | 描述                                               |
| -------- | -------------------------------------------------- |
| 年龄区间 | 18-35岁                                            |
| 特征     | 对玄学/命理/塔罗好奇，希望通过AI获取人生指导        |
| 痛点     | 线下算命不靠谱、缺乏便捷的命理咨询渠道             |
| 核心需求 | AI智能问事、八字命理、塔罗牌等玄学技能体验          |
| 期望场景 | 与AI对话获取命理分析、了解姻缘/事业/健康指引        |

### 2.3 文创爱好者

| 属性     | 描述                                               |
| -------- | -------------------------------------------------- |
| 年龄区间 | 20-40岁                                            |
| 特征     | 喜欢DIY手串等传统文化饰品，注重个性化表达          |
| 痛点     | 市面手串款式单一、不了解五行属性搭配、缺乏专业指导  |
| 核心需求 | 自主设计手串、浏览大师作品、了解珠材功效属性        |
| 期望场景 | 选择珠材自由搭配设计、获取寺庙加持、分享作品到朋友圈 |

## 5. 用户核心流程

### 5.1 预约祈福流程

```
首页
 └-> 找寺院 / 找师傅
     └-> 选择寺院 / 师傅
         └-> 查看详情 (temple-detail / master-profile)
             └-> 立即预约
                 └-> 选择服务 / 日期 / 时段 / 功德金
                     └-> 确认支付 (booking.html)
```

### 5.2 AI问事流程

```
首页 Tab
 └-> AI问事 (ai-divination.html)
     └-> 选择技能 (八字命理/姻缘测算/塔罗牌等)
         └-> AI对话
             └-> 查看分析结果
```

### 5.3 热门服务流程

```
首页
 └-> 热门服务 (选择类型: 祈福/供灯/上香/还愿/超度/开光/化太岁)
     └-> 服务列表 (service-xxx.html, 筛选)
         └-> 选择寺院服务
             └-> 立即预约
                 └-> 预约下单 (booking.html)
```

### 5.4 DIY手串完整流程

```
首页
 └-> DIY手串入口 (diy-bracelet.html)
     │
     ├── [路线A: 浏览灵感]
     │   └-> 设计广场
     │       └-> 设计详情 (diy-detail.html)
     │           └-> 使用此设计
     │               └-> 设计页 (diy-design.html, 已加载材料)
     │                   └-> 预览下单 (diy-order.html)
     │                       └-> 选择额外加持服务
     │                           └-> 立即制作 (支付)
     │
     └── [路线B: 自主设计]
         └-> 自己DIY
             └-> 设计页 (diy-design.html, 选择珠材/配饰/佛头)
                 └-> 完成设计
                     └-> 预览下单 (diy-order.html)
                         └-> 选择额外加持服务
                             └-> 立即制作 (支付)
                     └-> [可选] 继续设计 <──> 预览下单 (可循环)
```
