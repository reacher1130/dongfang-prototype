## 1. 产品定位

| 项目     | 内容                                                           |
| -------- | -------------------------------------------------------------- |
| 产品名称 | 问玄东方                                                       |
| 产品描述 | 一款融合佛教、道教传统文化与现代科技的移动互联网平台             |
| 目标用户 | 对佛教/道教文化感兴趣的信众、寻求心灵慰藉的用户、对玄学/命理好奇的年轻人 |
| 核心价值 | 连接信众与寺院/法师，提供在线祈福、AI问事、DIY手串定制等一站式服务 |

### 产品愿景

将中国传统文化与移动互联网技术深度融合，打造一个可信赖的佛道教文化服务平台，让用户随时随地便捷地接触玄学文化、获取命理指导、体验传统仪式。

### 目标用户画像

| 用户类型     | 年龄区间   | 特征描述                                           |
| ------------ | ---------- | -------------------------------------------------- |
| 信众用户     | 25-55岁    | 对佛教/道教有信仰，经常去寺庙祈福、供养           |
| 好奇用户     | 18-35岁    | 对玄学/命理/塔罗好奇，希望通过AI获取人生指导      |
| 文创爱好者   | 20-40岁    | 喜欢DIY手串等传统文化饰品，注重个性化表达          |

### 2.4 组件规范

#### 毛玻璃导航栏
- **顶部导航栏**: `rgba(28,18,16,0.92)` + `backdrop-filter: blur(12px)`，高度 44px，粘性定位
- **底部Tab栏**: `rgba(28,18,16,0.95)` + `backdrop-filter: blur(20px)`，固定定位，5个Tab
- **安全区域适配**: `padding-bottom: calc(8px + env(safe-area-inset-bottom, 0px))`

#### 底部Tab栏
- 5个Tab: 首页 / 对话 / AI问事 / 商城 / 我的
- 激活态: 朱砂红图标 + 文字
- 非激活态: `#8A7A6A` 灰色图标 + 文字

#### 卡片
- 背景: `#2A1E1A`
- 边框: `1px solid rgba(200,169,110,0.15)`
- 圆角: `12px`

#### 按钮
- **主按钮**: 渐变 `#C45A3C` -> `#D97B4A`，圆角 `10-12px`，高度 44px
- **副按钮**: 描边 `#C8A96E`，透明背景，金色文字

#### 标签药丸 (Tag Pill)
- 圆角: `9999px`，padding: `5px 14px`
- 激活态: `#C45A3C` 背景 + 白色文字
- 默认态: `#3A2C25` 背景 + `#8A7A6A` 文字

#### 水平滚动
- 隐藏滚动条 (`scrollbar-width: none`)
- 支持 scroll-snap 吸附
- `-webkit-overflow-scrolling: touch` 惯性滚动

#### 底部操作栏
- 固定定位: `position: fixed; bottom: 0; z-index: 100`
- 背景: `rgba(28,18,16,0.95)` + `backdrop-filter: blur(20px)`
- 安全区域适配

#### 圆点指示器
- 默认态: `6px` 圆形 + `#8A7A6A` 背景
- 激活态: `18px` 胶囊形 + `#C45A3C` 背景

### 5.4 开发注意事项

1. **`.design` canvas 不支持 Tailwind 颜色类**（如 `bg-bg-primary` 等），所有背景必须使用内联 `style` 属性
2. SVG手串预览采用内联SVG实现，支持动态珠位填充与空位展示
3. 毛玻璃效果依赖 `backdrop-filter`，需要 `-webkit-` 前缀兼容 Safari
4. 底部安全区域使用 `env(safe-area-inset-bottom)` 适配刘海屏
5. 所有页面均为移动端优先设计，推荐在 375-430px 视口下预览
